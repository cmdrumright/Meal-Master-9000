# Meal Master 9000

A webapp for creating weekly meal plans. Add foods, create meals, and add meal to the plan for the week.

## How to Run

### 1. Clone the repo and cd

```
git clone https://github.com/cmdrumright/Meal-Master-9000.git
cd Meal-Master-9000
```

### 2. Run the database

```
json-server -p 8088 -w database/database.json
```

### 3. Run React

```
npm run dev
```

### 4. Open webpage

http://localhost:5173/
